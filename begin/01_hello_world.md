# 01. 初识仓颉 (Hello World)

欢迎来到仓颉编程世界！编程不仅仅是写代码，更是与计算机对话的艺术。让我们从最简单的“问候”开始，感受仓颉的优雅。

## 1. 环境准备

在开始之前，请确保你已经安装了仓颉编译器 (`cjc`)。

验证安装：
```bash
cjc -v
```

## 2. 你的第一行魔法咒语

创建一个名为 `hello.cj` 的文件。我们不仅要打印 "Hello"，还要让计算机告诉我们当前的时间，赋予程序一点“生命力”。

```cangjie
import std.time.*

main() {
    println("你好，仓颉！")
    println("现在是编程的最佳时间。")

    // 简单的计算，展示程序不仅仅是打印文本
    let year = 2024
    println("欢迎来到 ${year} 年的仓颉世界！")
}
```

*   `main()`: 程序的灵魂入口。
*   `println`: 最常用的输出指令。
*   `${year}`: 字符串插值，让变量直接嵌入文本中。

## 3. 编译与施法 (运行)

将代码编译成可执行文件：

```bash
cjc hello.cj -o hello
```

运行它：

```bash
./hello
```

输出：
```text
你好，仓颉！
现在是编程的最佳时间。
欢迎来到 2024 年的仓颉世界！
```

恭喜！你已经成功唤醒了你的第一个仓颉程序。
