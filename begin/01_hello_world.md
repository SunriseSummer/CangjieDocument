# 01. 初识仓颉 (Hello World)

欢迎开启仓颉编程语言的学习之旅！本教程将带你编写并运行第一个仓颉程序。

## 1. 环境准备

确保你已经安装了仓颉编译器，并且配置好了 `CANGJIE_HOME` 环境变量，同时将 `bin` 目录添加到了系统 `PATH` 中。

验证安装：
```bash
cjc -v
```

## 2. 编写代码

创建一个名为 `hello.cj` 的文件，输入以下代码：

```cangjie
main() {
    println("你好，仓颉！")
}
```

*   `main` 是程序的入口函数。
*   `println` 是标准库提供的打印函数，用于输出信息并换行。
*   `.cj` 是仓颉源代码的扩展名。

## 3. 编译与运行

在终端中执行以下命令进行编译：

```bash
cjc hello.cj -o hello
```

*   `cjc` 是仓颉编译器。
*   `-o hello` 指定输出的可执行文件名为 `hello` (Windows 下会自动加上 .exe)。

运行程序：

```bash
./hello
```

输出：
```text
你好，仓颉！
```

恭喜你，你已经迈出了第一步！
