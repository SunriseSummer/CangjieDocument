# 元组类型 (Tuple)

元组 (`Tuple`) 是一种固定长度的数据结构，用于将多个不同类型的值组合成一个单一的复合值。

## 1. 定义与创建

使用圆括号 `()` 将多个值括起来，值之间用逗号分隔。

<!-- compile -->
```cangjie
let person: (String, Int64) = ("Alice", 30)
let point = (10, 20) // 类型推断为 (Int64, Int64)
```

## 2. 访问元素

### 使用索引
元组不支持像数组那样的动态下标 `[]`，而是通过直接解构或特定的访问方式（注：标准文档通常推荐解构）。

### 使用解构 (Destructuring)
这是访问元组最常用、最优雅的方式。

<!-- verify -->
```cangjie
main() {
    let point = (10, 20)

    // 将元组的值分别赋给 x 和 y
    let (x, y) = point
    println("x=${x}, y=${y}")

    // 使用 _ 忽略不需要的元素
    let (onlyX, _) = point
    println("onlyX=${onlyX}")
}
```

## 3. 特性

- **类型安全**: `(Int64, Int64)` 和 `(Int64, String)` 是完全不同的类型。
- **长度固定**: 一旦创建，无法添加或删除元素。
- **比较**: 如果两个元组的元素类型相同且对应元素都支持比较（如 `==`），则这两个元组也可以比较。

<!-- verify -->
```cangjie
main() {
    let t1 = (1, "a")
    let t2 = (1, "a")
    println(t1 == t2) // true
}
```

> **💡 提示**: 元组非常适合用于函数需要返回多个值的场景。
