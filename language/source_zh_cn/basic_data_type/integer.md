# 整数类型

仓颉语言提供了丰富多样的整数类型，分为**有符号 (Signed)** 和 **无符号 (Unsigned)** 两大类，以满足不同场景下的数值计算需求。

## 1. 类型概览

### 有符号整数
用于表示正数、负数和零。

| 类型 | 位宽 | 说明 |
| :--- | :--- | :--- |
| **`Int8`** | 8-bit | 字节型整数 |
| **`Int16`** | 16-bit | 短整数 |
| **`Int32`** | 32-bit | 标准整数 |
| **`Int64`** | 64-bit | 长整数（**推荐默认使用**） |
| **`IntNative`** | 平台相关 | 取决于运行平台（32位或64位），常用于索引或内存大小 |

### 无符号整数
仅用于表示非负数（0 和正数）。

| 类型 | 位宽 | 说明 |
| :--- | :--- | :--- |
| **`UInt8`** | 8-bit | 字节值，常用于处理二进制数据 |
| **`UInt16`** | 16-bit | - |
| **`UInt32`** | 32-bit | - |
| **`UInt64`** | 64-bit | - |
| **`UIntNative`** | 平台相关 | 取决于运行平台 |

### 取值范围

| 类型 | 最小值 | 最大值 |
| :--- | :--- | :--- |
| `Int8` | -128 | 127 |
| `Int16` | -32,768 | 32,767 |
| `Int32` | -2,147,483,648 | 2,147,483,647 |
| `Int64` | -9,223,372,036,854,775,808 | 9,223,372,036,854,775,807 |
| `UInt8` | 0 | 255 |
| `UInt32` | 0 | 4,294,967,295 |
| `UInt64` | 0 | 18,446,744,073,709,551,615 |

> **💡 最佳实践**: 在一般场景下，**首选 `Int64`**。它的范围足够大，且仓颉的整数字面量默认推断为 `Int64`，使用它可以减少不必要的类型转换。

## 2. 整数类型字面量

### 进制表示
支持四种进制写法，并允许使用下划线 `_` 作为分隔符以增强可读性。

| 进制 | 前缀 | 示例 | 对应十进制 |
| :--- | :--- | :--- | :--- |
| **十进制** | 无 | `24`, `1_000` | 24, 1000 |
| **二进制** | `0b` / `0B` | `0b11000` | 24 |
| **八进制** | `0o` / `0O` | `0o30` | 24 |
| **十六进制** | `0x` / `0X` | `0x18`, `0xFF` | 24, 255 |

### 类型后缀
默认情况下，整数字面量（如 `100`）被推断为 `Int64`。可以使用后缀显式指定类型。

| 后缀 | 类型 | 示例 |
| :--- | :--- | :--- |
| `i8`, `u8` | `Int8`, `UInt8` | `100i8`, `0xFFu8` |
| `i16`, `u16` | `Int16`, `UInt16` | `30000i16` |
| `i32`, `u32` | `Int32`, `UInt32` | `123i32` |
| `i64`, `u64` | `Int64`, `UInt64` | `100i64` (默认), `0x10u64` |

<!-- compile -->
```cangjie
let a = 100i8    // Int8
let b = 0x10u64  // UInt64
let c = 0o77i32  // Int32
```

## 3. 字节字面量 (Byte Literals)

`UInt8` 类型有一个别名 `Byte`。仓颉提供了专门的**字符字节字面量**，方便用 ASCII 字符表示字节值。

- **语法**: `b'字符'`
- **示例**:

<!-- compile -->
```cangjie
let a = b'x'      // 对应 ASCII 码 120 (UInt8)
let b = b'\n'     // 换行符，对应 10 (UInt8)
let c = b'\u{78}' // 十六进制转义，对应 120 (UInt8)
```

## 4. 支持的操作

整数支持常见的数学和位运算：
- **算术**: `+`, `-`, `*`, `/`, `%`, `**` (幂运算)
- **位运算**: `&`, `|`, `^`, `~`, `<<`, `>>`
- **比较**: `==`, `!=`, `<`, `>`, `<=`, `>=`

> 详见 [操作符](../Appendix/operator.md) 章节。
