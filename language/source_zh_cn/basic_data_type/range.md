# 区间类型 (Range)

区间类型用于表示一个数值序列的范围，常用于 `for` 循环遍历或数组切片。

## 1. 创建区间

仓颉支持两种区间操作符：

### 左闭右开 `..`
包含起始值，**不包含**结束值。

- **语法**: `start..end`
- **示例**: `0..5` 表示 `0, 1, 2, 3, 4`。

### 左闭右闭 `..=`
包含起始值，**包含**结束值。

- **语法**: `start..=end`
- **示例**: `0..=5` 表示 `0, 1, 2, 3, 4, 5`。

<!-- verify -->
```cangjie
main() {
    for (i in 1..4) {
        print(i) // 输出 123
    }
    println("")

    for (i in 1..=4) {
        print(i) // 输出 1234
    }
}
```

## 2. 步长 (Step)

默认情况下，区间的步长为 1。如果需要其他步长，可以使用 `step` 属性（或相关函数，具体视标准库而定，但在基础语法中通常结合 `for` 循环或迭代器使用）。

> 注：在基本的区间字面量中不直接包含步长语法，通常通过迭代器方法实现。

## 3. 这里的用法

### 用于切片
在数组切片中，区间用于指定子数组的范围。
- 支持省略语法：`start..` (从 start 到末尾), `..end` (从开头到 end)。

<!-- compile -->
```cangjie
let arr = [0, 1, 2, 3, 4]
let slice1 = arr[1..3] // [1, 2]
let slice2 = arr[2..]  // [2, 3, 4]
let slice3 = arr[..2]  // [0, 1]
```

### 用于判断
可以使用 `contains` 方法（或 `match`）判断一个值是否在区间内。

<!-- verify -->
```cangjie
main() {
    let r = 1..10
    if (r.contains(5)) {
        println("5 in range")
    }
}
```
