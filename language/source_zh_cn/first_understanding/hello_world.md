# 运行第一个仓颉程序

万事俱备，让我们开始编写并运行第一个仓颉程序吧！

## 1. 编写代码

请在您喜欢的目录下新建一个名为 `hello.cj` 的文本文件，并输入以下代码：

<!-- verify -->

```cangjie
// hello.cj
main() {
    println("你好，仓颉")
}
```

### 代码解析

- **注释**: 第一行 `// hello.cj` 是单行注释。仓颉支持 `//` 单行注释和 `/* ... */` 多行注释。
- **main 函数**: `main()` 是仓颉程序的入口点。程序启动时，会首先执行 `main` 函数体内的代码。注意，在仓颉中定义 `main` 函数时不需要 `func` 关键字。
- **println**: 这是一个内置函数，用于将文本输出到控制台并换行。

## 2. 编译程序

打开终端（Terminal）或命令行窗口，进入文件所在目录，执行以下命令进行编译：

```bash
cjc hello.cj -o hello
```

- `cjc`: 仓颉编译器命令。
- `hello.cj`: 源代码文件。
- `-o hello`: 指定输出的可执行文件名为 `hello` (Windows 下通常建议指定为 `hello.exe`)。

> **💡 提示**: 如果是 Windows 平台，建议使用命令 `cjc hello.cj -o hello.exe`。

## 3. 运行程序

编译成功后，当前目录下会生成一个可执行文件。

**Linux / macOS:**
```bash
./hello
```

**Windows:**
```bash
hello.exe
```

运行后，您将看到终端输出：
```text
你好，仓颉
```

## 进阶：体验插值字符串

让我们稍微修改一下代码，体验仓颉方便的**插值字符串**功能。它可以让您在字符串中轻松嵌入变量值。

将 `hello.cj` 修改为：

<!-- verify -->

```cangjie
main() {
    let name = "仓颉"
    println("你好，${name}")
}
```

这里我们定义了一个变量 `name`，并在字符串中使用 `${name}` 来引用它。编译并再次运行，您将看到相同的输出结果，但代码的逻辑更加灵活了！
