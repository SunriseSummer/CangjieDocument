# 第一章：塔之入口 (Hello World)

> 你站在传说中的“仓颉魔塔”脚下。古老的石门紧闭，门上刻着一行闪光的符文。传说只有能与魔塔“对话”的人，才能开启大门，获得无上的编程智慧。

## 1. 唤醒守护灵

为了开启大门，你需要向魔塔的守护灵发送第一条讯息。这不仅是一句问候，更是你与这个世界建立连接的契机。

在你的魔法卷轴（编辑器）中，写下这行符文：

```cangjie
// file: magic_tower.cj
main() {
    println("魔塔守护者，我是前来挑战的勇者！")
}
```

*   **符文解析**：
    *   `main()`: 这是所有魔法的起源，你的灵魂入口。
    *   `println`: “咏唱”指令，将你的意志投射到现实（屏幕）。

## 2. 激活符文 (编译与运行)

现在，举起你的法杖（终端），执行以下命令来激活符文：

```bash
cjc magic_tower.cj -o entry_spell
./entry_spell
```

当你看到屏幕上浮现出：
```text
魔塔守护者，我是前来挑战的勇者！
```
大门轰然开启，一道光芒将你吸入塔内。欢迎来到第一层！
